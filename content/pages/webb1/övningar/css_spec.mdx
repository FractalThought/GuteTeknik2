# Förberedelser

Skapa ett HTML-dokument som heter `css_spec.html`. Lägg in grunden till sidan på samma sätt som vi gjort tidigare (med ! och Enter-tangenten) och ge dokumentet titeln *CSS spec*.

# 1: Någonting att stila

Kopiera följande HTML-kod och lägg den i din body-tagg.

<Notice>

Ovanstående kod innehåller tre nya typer av element;
- **div**: Divider, används för att gruppera element tillsammans för stilning.
- **li**: List item, listobjekt. I en punktlistan är ett list item en enskild punkt.
- **ul**: Unordered list (punktlista), en behållare för list items. Visar vilka listobjekt som tillhör vilken lista.

</Notice>

# 2: Style-taggen

I `<head>`-taggen, skapa en `<style>`-tagg. Det är i denna taggen du ska skriva all din CSS. **I den här övningen ska du inte skriva någon HTML-kod alls, endast CSS**.

# 3: Stilandet

Följ dessa instruktioner om hur hemsidan ska stilas.

### Stila så all text på sidan använder följande deklaration:

```css
{font-family: Verdana, Arial, sans-serif;}
```

### Ge alla element med klassen intro följande deklaration:

```css
{font-weight: bold;}
```

### Stila endast diven som innehåller information om Caleb med följande deklaration:

```css
{background: lightcoral;}
```

### Gör samma för diven som innehåller information om Jester, men låt bakgrundsfärgen vara `lightblue` istället.


### Stila alla paragrafer som ligger direkt i body-taggen med följande deklaration:

```css
    {font-style: italic;}
```

### Stila alla paragrafer som ligger i caleb och jester-divarna med följande deklarationer:

```css
{color: indigo;}
```

### Varje medlem i medlemslistan har en klass, förutom Fjord och Veth som har två klasser.

Stila klasserna **wizard**, **warlock**, **paladin**, och **rogue** (i den ordningen) med varsin färg. **Notera vilken färg karaktärens namn får.**
