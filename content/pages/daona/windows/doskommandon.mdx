Microsoft Disk Operating System, mer känt som MS-DOS eller bara DOS, var standard i datorer på 80-talet och början av 90-talet. Det är ett helt textbaserat operativsystem, ofta kallad konsolbaserat operativsystem eftersom . Sista versionen av DOS kom år 2000, men DOS lever vidare i datorer som använder Windows i form av Kommandotolken (Command Prompt). Kommandona som används för att styra Kommandotolken kallas oftast för DOS-kommandon (DOS commands).

# Starta Kommandotolken

För att starta kommandotolken, öppna Startmenyn i Windows (tryck på Windows-tangenten på tangentbordet) och skriv (sök) `cmd`, kommandotolken borde då dyka om som förslag.

## Alternativt sätt att starta:

Använd kortkommandot Windows+R (Windowstangenten och R) för att få upp Kör (Run) och skriv in `cmd` och tryck på Ok-knappen.

# DOSkommandon

Med DOSkommandon kan man göra det mest som vi annars kan göra med Utforskaren, den applikation vi använder för att se och använda filer.

Ett DOSkommando fungerar så att vi använder ett kommando följt av parametrar och växlar.
Kommandot bestämmer vad för typ av sak vi ska göra (till exempel skapa en folder eller flytta en fil) och parametrarna bestämmer vad vi gör med det kommandot (till exempel folderns namn, filens namn, vart filen ska flyttas till).
