Windows har mängder av olika kortkommandon (tangentbordsgenvägar/keyboard shortcuts) som gör att vi i princip kan styra hela datorn med endast ett tangentbord.

Kortkommandon kan skynda på och förenkla användandet av datorn. Vi ska gå igenom två typer av kortkommandon; **Programspecifika** och **Operativsystemspecifika**. Operativsystemspecifika är kommandon som Windows använder oavsett vilket program du kör just då, och Programspecifika är för det aktuella programmet som har fokus eller har tillgång till att lyssna efter tangenttryck.

# Kort om fönster och aktiva program

Namnet Windows kommer av konceptet att man får ett fönster in till ett program. När ett program är igång har den möjlighet att lyssna från input från användaren, men oftast är det begränsat till när något av programmmets fönster har fokus.

Du märker om ett program har fokus genom att det är markerat i Aktivitetsfältet. Inuti ett program kan olika fönster eller delar ha fokus, men oftast bara ett i taget. Fokus bestämmer alltså vad användarens input styr.

# Operativsystemsspecifika

## Windowstangenten

Windowstangenten öppnar Startmenyn. Börjar man skriva när startmenyn är uppe så startar man sökfunktionen i Windows som i första hand söker efter program att starta, i andra hand filer på datorn, och i tredje hand en internetsökning.

## Windowstangenten+E

Öppnar Utforskaren.

## Windowstangenten+D

Visar Skrivbordet.

## Windowstangenten+R

Startar Kör...-programmet som kan användas för att starta andra program, till exempel **cmd** för Kommandotolken eller **dxdiag** för diagnostikverktyget för DirectX.

## Windowstangenten+Printscreen

Tar en skärmdump (screenshot) och sparar det till en fil i Bilder/Skärmbilder-foldern.

## Windows+Shift+S

Öppnar del av skärmklippsverktyget som gör det lätt att ta en skärmdump på en del av skärmen. OBS! Detta fungerar på liknande sätt som att bara använda Printscreen-knappen, alltså att du behöver klistra in bilden i ett bildredigeringsprogram för att spara bilden.

## Windowstangenten+. (punkt)

Öppnar Emoji-panelen så vi kan skriva in Emojis, smileys, och särskilda tecken.

## Alt+Tab

Byter fönster till det förgående fönstret som hade fokus.

Håll inne Alt och tryck på Tab en gång för att öppna en växlingsvy där du kan tabba (tryck på tab) för att komma till ett fönster du vill skifta fokus till. Släpp Alt för att gå till det valda fönstret.

## Windows+Tab

Liknar Alt+Tab, men öppnar en vy som gör att du kan öppna fler virtuella skrivbord och skifta mellan dem. Använd Enter-knappen för att välja, och Exit-knappen för att avbryta. Använder du musknappen kan du hovra över och klicka för att välja.

## Windowstangenten+Pilknapp

Med pilknapparna kan du placera det aktiva fönstret (fönstret som har fokus) till höger, vänster, minimera det, maximera det, osv. Håll in Windowstangenten och tryck sedan på en piltangent och se fönstret hoppa i en riktning.

## Alt+F4 eller Ctrl+W

Stänger det aktiva fönstret.

## Ctrl+Alt+Delete

Kallas även "Trefingersalut" eller "Säkerhetstangenterna". Ursprunliga funktionen av Ctrl+Alt+Delete var att starta om datorn. Senare, i Windows 9x-system användes den för starta Aktivitetshanteraren.

Ctrl Alt Delete öppnar numera Windows Security, en särskild skärm där användaren kan logga ut, starta om, växla användare, låsa datorn, starta hjälpmedel, osv. Därifrån kan man även starta Aktivitetshanteraren.

# Programspecifika

## Ctrl+C, Ctrl+X, Ctrl+V

Ctrlknappen tillsammans med C, X, och V styr Urklippning, där vi kan klippa ut en bit text eller något annat som är markerat i programmet med Ctrl+X, och klistra in det (effektivt flytta objektet) med Ctrl+V. Ctrl+C kopierar istället för att klippa ut.

Vissa program har även Ctrl+D vilket betyder Duplicera. Ctrl+D kopierar det markerade objektet och klistrar in det direkt på samma plats/folder. OBS! I Utforskaren och andra program kan Ctrl+D ha samma funktion som Delete-knappen, och tar alltså bort det markerade objektet.

## CTRL+Z och Ctrl+Y

Ctrl+Z används i de flesta program som genvägen för Undo (ångra, göra ogjort) som tar bort den senaste ändringen du gjorde i programmet. Ctrl+Y används oftast för Redo (gör om) för att återställa ändringen du tog bort med Undo.

## Ctrl+A

Används oftast för att "markera allt". I en textredigerare markeras all text, i utforskaren markeras alla foldrar och filer.

## Ctrl+P

Används oftast för att skriva ut/starta utskriftsfunktionen i programmet. I program så som VSCode där utskrift inte är lika vanligt, används **Ctrl+P** för att öppna en specialiserad kommandopalett där man kan skriva in ett kommando för att till exempel byta vilken flik som har fokus, eller utföra handlingar och kommandon inuti programmet som kanske saknar eget kortkommando.

## Ctrl+F / Ctrl+H (hitta/sök)

Används oftast för att söka (Find/Hitta). Ctrl+F används i de flesta program, Ctrl+H används i bland annat Microsoft Word eftersom Ctrl+F används där istället för att Fetmarkera något. De flesta program som har möjlighet att Fetmarkera text använder istället Ctrl+B (från engelskans Bold). Word använder Ctrl+F på grund av beslut som togs för längesedan och nu är det svårare att göra en sådan ändring då det kräver ominlärning och kan leda till frustration hos användarna.

## F11 (helskärmsläge)

F11 är oftast kommandot för helskärmsläge - ett kommando för att få det aktiva fönstret att ta upp hela skärmen.

## Alt

Om du trycker på Alt i de flesta program så kommer huvudmenyn i programmet markeras och du kan med piltangenterna gå igenom vad du vill starta från huvudmenyn. Den markerar även en tangent som man kan använda för att gå dit direkt.

## Ctrl+Tab (ibland Shift+Tab)

I program som har flera tabbar (flikar på svenska) brukar Ctrl+Tab göra att man gå igenom tabbarna. Ibland används Shift+Tab istället.

Det är även vanligt att man kan använda Ctrl+siffertangenterna för att gå till 1a-8e tabben, eller sista tabben med 9.

### Ny flik

Olika program kan ha olika kortkommandon för att öppna en ny flik eller nytt fönster, men vanligast är Ctrl+N, Shift+N, eller Alt+N. Webbläsare har vanligtvis speciella kortkommandon för att öppna ett nytt fönster i "Inkognito-läge" - i Chrome är det Ctrl+Shift+N och i Firefox är det Ctrl+Shift+P.
# Navigera ett program med bara tangentbordet

**Tab-tangenten** är generellt gjord för tillgänglighet och används för att "tabba igenom" fokus för programmets element. Med Tab kan du i till exempel en webbläsare växla mellan vilken container, länk, eller input som har fokus. Detta gör att du kan navigera en hemsida med endast tangentbordet. Det element som har fokus bör vara markerat på något sätt, ofast med en streckad ram runt elementet.

Som standard går fokuset nedåt när man tabbar, i en lista som kan bestämmas av skaparen av webbläsaren (kallas att man styr Tab index). **Shift+Tab** gör att man går uppåt i listan istället.

För att högerklicka används en speciell tangent som vanligtvis ligger på höger sida om **mellanslagstangenten**, mellan **Alt Gr** och **Right Ctrl**. Tangenten heter Meny-tangenten och styr vanligast *Kontextmenyn*, även kallad *Högerklicksmenyn*.

För att öppna en länk i en ny fil används oftast **Ctrl+Enter**.

När du navigerar en hemsida med endast tangentbordet så kan man använda piltangenterna för att skrolla. Detta påverkar inte fokus.


## Textmarkörsläge (F7)

Att markera text på en hemsida är enkelt men en mus eller touchpad - du drar muspekaren till texten du vill markera, och klickar och drar för att markera texten. Om man bara har ett tangentbord behöver man dock aktivera något som kallas Textmakörsläge. Då kommer en textmarkör att dyka upp på sidan, och du kan använda piltangenterna och Shift för att markera text. **Ctrl+C** kopierar texten du har markerat.

<Notice half="true">

Protip: I de flesta program, **Ctrl+Shift+Piltangent** markerar ord för ord istället för tecken för tecken.

</Notice>
