Övningar behöver inte lämnas in.

## Övning 1
1. Skapa en variabel som heter **firstName** med `"Alfred"` som värde, eller ditt eget namn.
2. Skriv ut variabelns värde i konsolen.

<PracticeContainer>

```js
let firstName = "Alfred";
console.log(firstName);
```

</PracticeContainer>


## Övning 2
1. Ändra så att det konsolen skriver ut blir `"Hej Alfred"`.

<PracticeContainer>

```js
console.log("Hej " + firstName);
```

</PracticeContainer>

## Övning 3
Skapa en variabel som heter **saldo** och sätt den till värdet `12`.

<PracticeContainer>

```js
let saldo = 12;
```

</PracticeContainer>

## Övning 4
I konsolen, skriv ut följande: `Saldo: `, samt variabeln **saldo**'s värde, och avsluta med `kr.`.

Till exempel så här: `Saldo: 12kr`.

<PracticeContainer>

```js
console.log("Saldo: " + saldo + "kr");
```

</PracticeContainer>

## Övning 5
Skapa tre variabler (**pay**, **subscriptionCost**, **snacksCost**) och sätt deras värden enlikt följande:
- **pay**: *20*
- **subscriptionCost**: *6*
- **snacksCost**: *2*

<PracticeContainer>

```js
let pay = 20;
let subscriptionCost = 6;
let snacksCost = 2;
```

</PracticeContainer>

## Övning 6
*Variabeln **pay** håller värdet på användarens lön/månad.*

1. Sätt **saldo** till sig själv, plus **pay**.
2. Skriv ut saldot på samma sätt som i Övning 4 (hedanefter, när du blir instruerad att skriva ut saldot, gör på detta sätt).

<PracticeContainer>

```js
saldo = saldo + pay;
console.log("Saldo: " + saldo + "kr.");
```

</PracticeContainer>

## Övning 7
*Variabeln **subscriptionCost** syftar till kostnaden/månad för att prenumerera på en tjänst (till exempel spel, streaming-tjänst, eller liknande).*

1. Skapa en variabel som visar kostnaden för att prenumerera på tre tjänster vars kostnad är lika med **subscriptionCost**. Namnet på variabeln väljer du själv, men tänk på att namnge den så beskrivande du kan.
2. Skriv ut i konsolen ett meddelande som säger vad kostnaden för en prenumerationstjänst kostar, och vad kostnaden för tre prenumerationstjänster kostar. Du ska använda variablernas värde i meddelande.

<PracticeContainer>

```js
let totalSubCost = subscriptionCost * 3; // Ett bättre sätt hade varit att sätta 3an till en variabel också, tex numberOfSubs
console.log("En prenumerationstjänst kostar " + subscriptionCost + "kr. 3 prenumerationer kostar därför " + totalSubCost + "kr.");
```

</PracticeContainer>

## Övning 8
1. Sätt **saldo** till sig själv, minus variabeln du skapade för totala prenumerationskostnaden.
2. Skriv ut saldot.

<PracticeContainer>

```js
saldo = saldo - totalSubCost;
console.log("Saldo: " + saldo + "kr.");
```

</PracticeContainer>

## Övning 9
*Variabeln **snacksCost** är kostnaden för att köpa en bit snacks (chokladkaka, godis eller chipspåse, eller liknande).*

1. Skapa en variabel (likt övning 7 ska du namnge variabeln själv) och dess värde ska vara antalet snacksbitar som användaren kan köpa med saldot som är kvar efter tidigare inkomster och utgifter.
2. Skriv ut ett meddelande i konsolen som berättar om antalet snacksbitar som användaren kan köpa.

<PracticeContainer>

```js
let numberOfSnacks = saldo / snacksCost;
console.log("Antalet snacks du kan köpa är " + numberOfSnacks);
```

</PracticeContainer>

## Övning 10

Undersök och experimentera med vad som händer med dina meddelanden till konsolen (kallade outputs) när du ändrar i dina variabler.

Försök göra så att du återställer alla variabler till sitt ursprungliga värde efter varje experiment.

### Gör minst följande:
1. Ändra dina variablers värde så att saldot efter användaren fått lön är `35`.
2. Undersök hur många snacksbitar som kan köpas om användaren prenumererar på `4` tjänster. Gör samma fast med `2` tjänster.
3. Ändra kostnaden för snacksbitar till `3`, hur många snacksbitar kan köpas?
4. Du får endast ändra variabeln **pay**. Med den ska du göra så att antalet snacksbitar som kan köpas i slutet blir `12`.
5. Ändra kostnaden för en prenumerationstjänst så att totala kostnaden för tjänsterna blir `36`. Undersök vad som händer med resten av dina meddelanden.

## Lösningen för Övning 1 till 9

```js

// Övning 1 och 2
let firstName = "Alfred";
console.log("Hej " + firstName); // Kom ihåg mellanslaget i "Hej "

// övning 3 och 4
let saldo = 12;
console.log("Saldo: " + saldo + "kr.");

// Övning 5
let pay = 20;
let subscriptionCost = 6;
let snacksCost = 2;

// Övning 6
saldo = saldo + pay;
/*
    Man kan också använda += så här:
    saldo += pay
*/
console.log("Saldo: " + saldo + "kr.");

// Övning 7
let totalSubCost = subscriptionCost * 3; // Ett bättre sätt hade varit att sätta 3an till en variabel också, tex numberOfSubs
console.log("En prenumerationstjänst kostar " + subscriptionCost + "kr. 3 prenumerationer kostar därför " + totalSubCost + "kr.");

// Övning 8
saldo = saldo - totalSubCost;
console.log("Saldo: " + saldo + "kr.");

// övning 9
let numberOfSnacks = saldo / snacksCost;
console.log("Antalet snacks du kan köpa är " + numberOfSnacks);
```
