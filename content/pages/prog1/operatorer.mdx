## Aritmetiska operatorer

```jsx
+ - * / %
```

Aritmetiska operatorer används för att utföra vanliga matematiska uträkningar.

## Tilldelande operatorer

```jsx
= += -= /= %=
```

Tilldelande (assignment) operatorer används för att tilldela värden till variabler.

## Förändringsoperatorer

```jsx
++ —-
```

Förändringsoperatorer används för att ändra värdet på en variabel med 1, antingen upp eller ner, beroende på operator.

## Relationsoperatorer (jämförelseoperatorer, comparison)

```jsx
== ≠ < > ≤ ≥
```

Relationsoperatorer (relationship eller comparison) används för att jämföra två värden med varandra. Resultatet från detta är antingen sant eller falskt, beroende på värdena.

## Logiska operatorer

```jsx
&& || !
```

Logiska operatorer används för att lägga ihop eller förändra andra operatorer.
