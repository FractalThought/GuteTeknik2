Introprojektet är första inlämningsprojektet för den här kursen.

# Syfte

Detta projekt har som syfte att testa dina kunskaper för grunderna inom programmering.

Du behöver använda variabler, kontrollsatser, iterationssatser, och arrayer.

# Bedömningspunkter

Vad behöver de uppvisa?

Använda randomfunktionen
Lägga till saker till arrayer
Skriva ut arrayen

Kalle har svårt att välja vad han vill göra härnäst.
Använd random funktionen för att välja aktivitet.

Lustiga huset ändrar layout? Labyrint? Oh! Generera en labyrint?

```javascript
let antalDagar = 1
let dagslon = 0.01,
  totaltBelopp = 0.01
let onskat = 1000000.0

while (totaltBelopp < onskat) {
  antalDagar++ //antalDagar=antalDagar+1;
  dagslon *= 2 //dagslon=dagslon*2;
  totaltBelopp += dagslon //totaltBelopp=totaltBelopp+dagslon;
  console.log(" Dag " + antalDagar + " Dagslön = " + dagslon)
}
console.log("Du blir miljonär efter " + antalDagar + " dagar")
```
