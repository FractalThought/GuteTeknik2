Gör övningarna i Visual Studio Code. Skapa en ny fil, namnge den *controlExcercises.js* och gör övningarna i den filen.

<Notice>
I alla övningarna ska du testa att ändra variablernas värden för att testa att ifsatserna fungerar. Har du till exempel en ifsats som kollar om en variabels värde är 18 eller mer så ska du testa koden med följande;
1. Minst en gång när variabelns värde är 18
2. Minst en gång när variabelns värde är mer än 18.
3. Minst en gång när variabelns värde är mindre än 18.
</Notice>

Övningar behöver inte lämnas in.

# Grunden
Klistra in följande kod i din fil och använd den för att lösa övningarna.

```javascript
let myName = "Alfred";
let age = 18;
let drivingAge = 18;
let pensionAge = 65;

// Följande kod är ett snabbare sätt att sätta flera variabler
// Man skriver bara en "let", gör en deklaration, och sätter en kommatecken för att kunna göra en till deklaration.
let saldo = 12,
  pay = 20,
  subscriptionCost = 6,
  snacksCost = 2;

let totalSubCost = subscriptionCost * amountOfSubs;

console.log(Alfred + " är " + age + "år.");
console.log("Man får köra bil när man är " + drivingAge);
console.log("När man är " + pensionAge + "år kan man gå i pension.");

```


# Övning 1

if(age == drivingAge) {

Skriv ut ett meddelande i konsolen om `age` är lägre, som säger att Alfred inte är i pensionsåldern än.
Bonus: Skriv ut hur många år Alfred har kvar till pensionsåldern.

# Övning 5
*Alfred gillar att använda olika prenumerationstjänster, men det får inte bli för dyrt. Om varje tjänst kostar 6kr eller mindre så vill han prenumerera på 3 tjänster.*

1. Skapa en ny variabel med namnet `amountOfSubs` och sätt dess värde till `0`.
2. I en ny kontrollsats, kolla om subscriptionCost är 6 eller mer. Om den är det, sätt `amountOfSubs` till 3.
3. Efter kontrollsatsen (efter ifsatsens `{}`), skapa en ny variabel med namnet `totalSubCost` och sätt den till `subscriptionCost * amountOfSubs`.
4. Skriv ut totala kostnaden för prenumerationstjänsterna (`totalSubCost`).

<PracticeContainer>

```javascript
let amountOfSubs = 0;
if(subscriptionCost >= 6) {
  amountOfSubs = 3;
}

let totalSubCost = subscriptionCost * amountOfSubs;

console.log("Totala kostnaden för Alfreds tjänster är " + "kr/månad.")
```

</PracticeContainer>

# Övning 6
*Om tjänsterna inte kostar 6kr eller mindre så vill Alfred bara prenumerera på 2 tjänster.*

Lägg till en else-sats till ifsatsen du skapade som sätter antalet prenumerationer (`amountOfSubs`) till 2. För detta behöver du gå upp innan där du skapar och skriver ut totala kostnaden.

Bonus: Tänk ut ett sätt som gör att du inte behöver använda else-satsen alls.

<PracticeContainer>

```javascript

let amountOfSubs = 0;
if(subscriptionCost >= 6) {
  amountOfSubs = 3;
} else {
  amountOfSubs = 2;
}

let totalSubCost = subscriptionCost * amountOfSubs;

console.log("Totala kostnaden för Alfreds tjänster är " + "kr/månad.")
```

</PracticeContainer>

<PracticeContainer title="Lösningen till bonusfrågan">

Istället för att skapa en else-sats, sätt `amountOfSubs` till 2 direkt när variabeln skapas. Eftersom kod i en ifsats inte körs som ifsatsens villkor inte är sant så sätts variabeln `amountOfSubs` aldrig om.

</PracticeContainer>

# Övning 7
*Alfred vill helst kunna köpa 8 snacksbitar, men om han inte kan det så vill han köpa 3 så han kan spara pengar till nästa månad.*

1. Se till att Alfred betalar sina prenumerationer genom att minska saldot med kostnaden för tjänsterna (`totalSubCost`).
2. Skapa sedan en ifsats och en tillhörande elsesats som kollar om Alfred kan köpa 8 snacksbitar. Om han kan det, minska saldot med kostnaden för 8 snacksbitar. Om han inte kan det, minska saldot med kostnaden för 3 snacksbitar.
3. Skriv ut antalet snacksbitar Alfred köper denna månaden och hur mycket pengar han har kvar i slutet av månaden.

Tips: Skapa en variabel du kan kolla mot. Kom ihåg att variabeln `snacksCost` håller kostnaden för en snacksbit.

<PracticeContainer>

```javascript
saldo -= totalSubCost;
let amountOfSnacks = saldo / snacksCost;

if(amountOfSnacks >= 8) {
  saldo -= 8 * snacksCost;
  console.log("Alfred köper 8 snacksbitar och har " + saldo + "kr kvar.");
} else {
  saldo -= 3 * snacksCost;
  console.log("Alfred köper 3 snacksbitar och har " + saldo + "kr kvar.");
}

```

</PracticeContainer>

# Övning 8
Dubbelkolla så att Alfred faktiskt har råd att köpa ens 3 snacksbitar. Annars ska han köpa så många snacksbitar han har råd med.

<PracticeContainer>

```javascript
saldo -= totalSubCost;
let amountOfSnacks = saldo / snacksCost;

if(amountOfSnacks >= 8) {
  saldo -= 8 * snacksCost;
  console.log("Alfred köper 8 snacksbitar och har " + saldo + "kr kvar.");
} else {
  if(amountOfSnacks >= 3)
  {
    saldo -= 3 * snacksCost;
    console.log("Alfred köper 3 snacksbitar och har " + saldo + "kr kvar.");
  }
  else {
    saldo -= amountOfSnacks * snacksCost;
    console.log("Alfred köper " + amountOfSnacks + " snacksbitar och har " + saldo + "kr kvar.");
  }
}

```

</PracticeContainer>
