# Övning 1

1. Skapa en variabel som heter `i` och ge den värdet **0**.
2. Skapa en while-loop vars villkor är att `i` är mindre än 5.
3. I while-loopen, plussa på värdet på `i` med 1, och skriv ut det i konsolen.

Hedanefter när du får instruktioner om att skapa en while-loop som ska loopas igenom ett antal gånger, skapa loopen på det här sättet.

<PracticeContainer>

```javascript
let i = 0;
while(i < 5) {
    i++; // Samma som i += 1; Eller i = i + 1;
    console.log(i);
}
```
