# Övning 3
*Du ska ta reda på vilka siffror som är jämt (jämt delbara med 2), och vilka som är udda, i siffrorna 0 till 20.*

Gör så här;

1. Skapa en loop som loopas 20 gånger.
2. Använd en if-elsesats för att kontrollera om talet är jämt eller udda.

Tips: För att kolla om något är jämt delbart med 2, använd modulo-operatorn (`%`).
Modulo-operatorn gör att vi kan kolla om det blir något över efter vi delat en siffra med en annan. Om det inte blir något över (resultatet blir 0) så är den siffran vi delar jämt delbar med siffran vi delar med.

Så här;

```javascript
if(10 % 2 == 0) {
  console.log("10 är jämt delbart med 2 (jämn siffra)");
}
```

# Övning 4
Alfred har en lön på 20 kronor, och en månadskostnad på 12 kr. Hur många månader tar det innan han har sparat ihop till 50kr eller mer, om han börjar med 12 kr i sitt sparkonto?

Skapa en loop som simulerar Alfreds ekonomi i en månad i taget (så varje gång loopen körs läggs Alfreds lön till saldot, och hans månadskostnader dras ifrån saldot). Skriv ut saldots värde i konsolen för varje månad.

När saldot är 50 eller mer ska loopen avslutas, och du ska skriva ut i konsolen hur många månader det tog.

Denna övning testar din problemlösningsförmåga: Bryt ner problemet, fundera kring vad du behöver göra, och testa dig fram! Det finns flera lösningar.

<PracticeContainer>

```javascript
let saldo = 12;
let pay = 20;
let monthlyExpenses = 12;

let months = 0;
while(saldo <= 50) {
  saldo += pay;
  saldo -= monthlyExpenses;
  console.log(saldo);
  months++;
}

console.log("Det tog " + months + " månader.");
```

</PracticeContainer>

# Övning 5
Multiplikationstabellen kan vara knepig att hålla i huvudet.
Med kod, räkna ut 3-ans multiplikationstabell (1 till 12), och skriv ut alla siffror på en rad, separerade med `", "` (kommatecken och mellanslag).

När du är klar ska det alltså se ut så här:

> 3, 6, 9, 12, 15, 18, 21, 24, 27, 30, 33, 36

Tips: Du kan skapa en tom sträng utanför loopen och använda den för att spara siffrorna.

<PracticeContainer>

```javascript

let multiTable = "";
for(let i = 1; i <= 12; i++) { // Skapa variabeln i, satt till 1; Kör loopen så länge i är mindre eller lika med 12; Lägg till 1 till i
  let result = 3 * i; // Hämta ut resultatet av multiplikationen
  multiTable += result; // Lägg till resultatet i strängen

  if(i != 12) { // Lite snyggare; Vi kollar om det inte är sista siffran
    multiTable += ", "; // Om det inte är sista siffran, lägg till ", "
  }

  console.log(multiTable);
}

```

</PracticeContainer>

# Övning 8
Använd en loop och metoden indexOf för att hitta antalet gånger som ordet `"spel"` förekommer i följande mening:

> Wiscon är ett spelkonvent som arrangeras av spelföreningen Östans Skuggor. Vi erbjuder intressanta, utvecklande och engagerande aktiviteter inom Spelhobbyn – för barn, ungdomar och vuxna. WisCon passar lika bra för dig som har hållit på med spel länge som för dig som är nyfiken och vill veta mer om olika spel. På Wiscon kan du testa figurspel, brädspel, kortspel, rollspel, och cosplay!

Denna övning kräver problemlösning och att pussla ihop de olika delarna du lärt dig hittills.

Detta vet du:
- Du skapar en for-loop med `for(let i = 0; i < 5; i++) { /* kör kod här */ }`
- indexOf kan hitta en sökterm på en sträng och hittar positionen på den första gången termen hittas. Om den inte hittar något så skickar den tillbaka värdet -1
- indexOf kan hitta positionen av nästa gång termen hittas med andra argumentet, som förslagsvis är förgående gång termen hittades, plus 1.
- Vi kan räkna med loopar med hjälp av någon form av loop counter (måste inte vara samma som loopens villkor).
- Man kan bryta en loop med nyckelordet `break`.
