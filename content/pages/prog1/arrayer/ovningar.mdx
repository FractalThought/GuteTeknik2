# Övning 3
Skapa en ny array med namnet `myOtherNumbers`, men låt den vara tom (skapa den så här; `let myOtherNumbers = [];`).
Med kod, fyll arrayen med **50st** värden som följer mönstret `förgående siffra + 3`. **Börja på siffran 2.**
**Du ska göra detta så effektivt som möjligt.**

<PracticeContainer>

Använd en loop för att lägga till värdena. Du behöver använda en variabel till som sköter värdet.

```javascript
let myOtherNumbers = [];
let currentNumber = 2;
for(let i = 0; i < 50; i++) {
    myOtherNumbers.push(currentNumber);
    currentNumber += 3;
}
```

</PracticeContainer>

# Övning 7: URL-adress

Att skriva, skicka, och sedan dela en sträng är hur URL-adresser fungerar i webbläsare.

En URL kan se ut så här:
`guteteknik.netlify.app/prog1/arrayer/intro`

Det som kommer efter första snedstrecken (`/`) innehåller information om navigeringen på webbplatsen och visar oss var vi är på hemsidan, där sista delen efter sista snedstrecken är den aktuella sidan vi är på nu, och de innan visar var den sidan ligger. Om någon del av dem ändras kommer vi till en annan plats, eventuellt till en sida som inte finns.

## Att göra:
Du får hela URLen: `guteteknik.netlify.app/prog1/arrayer/intro`
Från den ska du hämta ut alla delar av navigeringen (alltså alla delar efter det första snedstrecken) och spara dem i en array.
Skriv sedan ut varje del i konsolen var för sig.

<PracticeContainer title="Tips">

Du behöver först isolera endast navigeringen, så använd metoderna `indexOf` för att hitta första snedstrecket och `slice` för att hämta ut allt efter det, så här;

```javascript
let url = "guteteknik.netlify.app/prog1/arrayer/intro";
let nav = url.slice(url.indexOf("/")+1) // Tar ut endast navigeringen (prog1/arrayer/intro)
// Fortsätt här
```

</PracticeContainer>
