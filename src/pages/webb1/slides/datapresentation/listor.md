# Introduktion till Listor

---

## Rubriker och paragrafer

```html
<h1></h1>
<p>Detta är en paragraf</p>
```

---

## List-item

```html
<h2>Characters</h2>
<li>Iron Man</li>
<li>Captain America</li>

<li>Superman</li>
<li>Batman</li>
```

## List-items behöver kapslas in

1. **OL:** Ordered list

- **UL:** Unordered list

```html
<ol></ol>
<li>Superman</li>
<li>Wonder Woman</li>
<li>Batman</li>
<ul>
  <li>Iron Man</li>
  <li>Captain America</li>
  <li>Spider-Man</li>
</ul>
```
